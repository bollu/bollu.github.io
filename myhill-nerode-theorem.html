§ Myhill Nerode Theorem

  • Take a language LLL over an alphabet AAA.
  • Define x∈A⋆x \in A^\starx∈A⋆ to have a disginguishing extension from y∈A⋆y \in A^\stary∈A⋆iff there exists a s∈A⋆s \in A^\stars∈A⋆ such that either (a) xs∈L∧ys∉Lxs \in L \land ys \not \in Lxs∈L∧ys∈L, or xs∉L∧ys∈Lxs \not \in L \land ys \in Lxs∈L∧ys∈L
  • Said differently, given xxx and yyy, there suffix sss which can distinguish xxx and yyy using the membership oracle for LLL.
  • Now define x∼Lyx \sim_L yx∼L​y ( xxx is indistinguishable from yyy) iff there is no distinguishing extension between xxx and yyy with respect to LLL.
  • See that this is an equivalence relation:
  • (1) Reflexivity : xxx cannot be distinguished from xxx (using LLL), because any suffix sss cannot produce different outputs for xxx and xxx.
  • (2) Symmetry : If xxx cannot be disguished from yyy (using LLL), then yyy cannot be distinguished from xxx (using LLL).
  • (3) Transitivity : If xxx cannot be distinguished from yyy (using LLL) and yyy cannot be distinguished from zzz (using LLL), then xxx cannot be distinguished from zzz (using LLL). Intuition: What about yyy in this situation? It can't be indistinguishable from both xxx and zzz.
  • Proof of Transitivity: Suppose for contradiction xxx can be distinguished from zzz. There there is a suffix sss such that xs∈Lxs \in Lxs∈L while zs∉Lzs \not \in Lzs∈L (WLOG). Now what about ysysys? if ys∈Lys \in Lys∈L then we can distinguish yyyand zzz, contradicting assumption that yyy is indistinguishable from zzz. If y∉Ly \not \in Ly∈L then we can distinguish xxx from yyy contradicting assumption that xxx is indistinguishable from yyy.
  • Hence, being indistinguishable is an equivalence relation, denoted by x∼yx \sim yx∼y.
  • Myhill nerode says that the minimal DFA for a language LLL has as many states as there are equivalence classes for ∼L\sim_L∼L​.

§ Given DFA DDD of language LLL over AAA: ∼D\sim_D∼D​ implies ∼L\sim_L∼L​

  • Let LLL be a regular langugage over alphabet AAA and let DDD be a DFA (with finite number of states ∣D∣|D|∣D∣ in the DFA).
  • Partition the set of all strings A∗⋆A*\starA∗⋆ via the relation ∼D\sim_D∼D​: x∼Dyx \sim_D yx∼D​y iff xxx and yyy end at the same state when fed to DFA.
  • Thus will have ∣D∣|D|∣D∣ equivalence classes for ∼D\sim_D∼D​, one for each state of the DFA.
  • For any two strings such that x∼Dyx \sim_D yx∼D​y, given any suffix sss, we have that xs∼Dysxs \sim_D ysxs∼D​yssince we start at the same state at xxx (or yyy) and continue when we feed the suffix sss.
  • Thus, strings such that x∼Dyx \sim_D yx∼D​y are indistinguishable for the DFA.
  • So we have x∼Lyx \sim_L yx∼L​y, since on any extension, both xsxsxs and ysysys either belong or don't belong to LLL.

§ given language LLL over AAA: show that ∼L\sim_L∼L​ implies ∼D\sim_D∼D​.

  • Let LLL be a langugage over alphabet AAA such that ∼L\sim_L∼L​ has finitely many equivalence classes.
  • We will design DFA (called DDD)for LLL with as many states as equivalence classes.
  • The start state of DDD is the equivalence class of the empty string with respect to ∼L\sim_L∼L​.
  • At an equivalence class/state TTT, given character c∈Ac \in Ac∈A, we move to T⋄cT \diamond cT⋄c (extend TTTby ccc).

§ DFA needs at least ∣A/∼L∣|A/\sim_L|∣A/∼L​∣ states

  • Let n≡∣A/∼L∣n \equiv |A/\sim_L|n≡∣A/∼L​∣ be the number of equivalence classes of ∼L\sim_L∼L​.
  • Suppose a DFA DDD recongizes LLL and has fewer than ∣A/∼L∣|A/\sim_L|∣A/∼L​∣ states.
  • Let x1,x2,…xnx_1, x_2, \dots x_nx1​,x2​,…xn​ be strings from different equivalence classes of LLL.
  • Push these strings through DDD. Some two strings xi,xjx_i, x_jxi​,xj​ must land on the state state d∈Dd \in Dd∈Dof DDD (by pigeonhole).
  • We must have xix_ixi​ and xjx_jxj​ distinguishable, since they come from different equivalence classes. So the DFA must accept one and reject the other.
  • But the DFA can't tell the difference between xix_ixi​ and xjx_jxj​ since they landed on the same state! So the DFA will accept or reject both.
  • Thus, we have a contradiction from the assumptions (a) DDD has fewer states than nnn and (b) DDD recognizes LLL.
  • Thus the DFA needs at least ∣A/∼L∣|A/\sim_L|∣A/∼L​∣ states.

§ The two imply DFA minimization

  • We have seen that every DFA for LLL needs at least ∣A/∼L∣|A/\sim L|∣A/∼L∣ states.
  • Now starting from LLL, we can build an automata D⋆D^\starD⋆ such that ∣D⋆∣|D^\star|∣D⋆∣ is exactly ∣A/∼L∣|A/\sim L|∣A/∼L∣.
  • Thus the automata D⋆D^\starD⋆ is a (the) minimal automata for LLL.
