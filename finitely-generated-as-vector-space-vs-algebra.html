§ Finitely generated as vector space v/s algebra:

  • To be finitely generated as a vector space over KKK from a generating set SSS means that we take elements of the form ∑ikisi\sum_i k_i s_i∑i​ki​si​, or abbreviated, elements of the form ∑KS\sum KS∑KS
  • To be finitely generated as a KKK algebra from a generating set SSS, we take elements of the form ∑ikisi+∑ijkijsisj+…\sum_i k_i s_i + \sum_{ij} k_{ij} s_i s_j + \dots∑i​ki​si​+∑ij​kij​si​sj​+…. To abbreviate, elements of the form ∑C+CS+CS2+CS3⋯=C/(1−S)\sum C + CS + CS^2 + CS^3 \dots = C/(1-S)∑C+CS+CS2+CS3⋯=C/(1−S).
As a trivial example, consider K[X]K[X]K[X]. This is not finitely generated as a vector space since it doesn't have a finite basis: the obvious choice of generating set {1,X,X2,… }\{ 1, X, X^2, \dots \}{1,X,X2,…} is not finite. It is finitely generated as a KKK-algebra with generating set {X}\{ X \}{X}.
