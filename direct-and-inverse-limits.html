§ Direct and Inverse limits

§ Direct limit: definition

A direct limit consists of injections A1→A2→…A_1 \rightarrow A_2 \rightarrow \dotsA1​→A2​→…. It leads to a limit object LLL, which as a set is equal to the union of all the AiA_iAi​. It is equipped with an equivalence relation. We can push data "towards" the limit object, hence it's a "direct" limit. So each element in AiA_iAi​ has an equivalence class representative in LLL.

§ Direct limit: prototypical example

§ SnS_nSn​

We can inject the symmetric groups S1→S2→…S_1 \rightarrow S_2 \rightarrow \dotsS1​→S2​→…. However, we cannot project back some permutation of S2S_2S2​ (say) to S1S_1S1​: if I have (2,1)(2, 1)(2,1) (swap 2 and 1), then I can't project this back into S1S_1S1​. This is prototypical; in general, we will only have injections into the limit, not projections out of the limit.

§ Prufer group

Here, the idea is to build a group consisting of all the pnp^npnth roots of unity. We can directly """define""" the group as:
P(q)∞≡{exp(2πk/qn):∀n,k∈N, 0≤k≤qn} P(q)^\infty \equiv \{ \texttt{exp}(2\pi k /q^n) : \forall n, k \in \mathbb N, ~ 0 \leq k \leq q^n \} P(q)∞≡{exp(2πk/qn):∀n,k∈N, 0≤k≤qn}
That is, we take q1q^1q1th roots of unity, q2q^2q2th roots of unity, and so on for all n∈Nn \in \mathbb Nn∈N. To build this as a direct limit, we embed the group Z/qnZZ/q^n ZZ/qnZ in Z/qn+1ZZ/q^{n+1}ZZ/qn+1Z by sending: the qnq^nqn th roots of unity to qn+1q^{n+1}qn+1th roots of unity raised to the power qqq. An example works well here.
  • To embed Z/9ZZ/9ZZ/9Z in Z/27ZZ/27ZZ/27Z, we send:
  • 2π1/92 \pi 1 /92π1/9 to 2π1/9×(3/3)=2π3/272 \pi 1/9 \times (3/3) = 2 \pi 3 / 272π1/9×(3/3)=2π3/27.
  • 2π2/92 \pi 2 /92π2/9 to 2π6/272 \pi 6/272π6/27
  • 2π3/92 \pi 3 /92π3/9 to 2π9/272 \pi 9 / 272π9/27
  • 2πk/92 \pi k / 92πk/9 to 2π(3k)/272 \pi (3k)/272π(3k)/27
  • This gives us a full embedding.
The direct limit of this gives us the prufer group. We can see that the prufer group is "different" from its components, since for one it has cardinality N\mathbb NN. For another, all subgroups of the prufer group are themselves infinite. The idea is to see that:
  • Every subgroup of the prufer group is finite.
  • By Lagrange, |prufer|/|subgroup| = |quotient|. But this gives us something like infinite/finite = infinite.
To see that every subgroup HHH of the prufer group is finite, pick an element ooo outside of the subgroup HHH. This element ooo will belong to some Z/qkZZ/q^kZZ/qkZ for some k∈Zk \in \mathbb Zk∈Z (since the direct limit has an elements the union of all the original elements modulo some equivalence). If the subgroup HHH does not have ooo (and thus does not contain Z/qkZZ/q^kZZ/qkZ), then we claim that it cannot contain any of the larger Z/qk+δZZ/q^{k+\delta}ZZ/qk+δZ. If it did contain the larger Z/qk+δZ/q^{k + \delta}Z/qk+δ, then it would also contain Z/qkZ/q^kZ/qk since we inject Z/qkZ/q^kZ/qk into Z/qk+δZ/q^{k+\delta}Z/qk+δ when building the prufer group. Thus, at MAXIMUM, the subgroup HHH can be Z/qk−1ZZ/q^{k-1}ZZ/qk−1Z, or smaller, which is finite in size. Pictorially: 
...         < NOT in H
Z/q^{k+1}Z  < NOT IN H
Z/q^kZ      < NOT IN H
---------
...         < MAYBE IN H, FINITE
Z/q^2Z      < MAYBE IN H, FINITE
Z/qZ        < MAYBE IN H, FINITE
The finite union of finite pieces is finite. This HHH is finite.

§ Stalks

Given a topological space (X,T)(X, T)(X,T) and functions to the reals on open sets F≡{U→R}F \equiv \{ U \rightarrow \R \}F≡{U→R}, we define the restricted function spaces F∣U≡{FU:U→R:f∈F}F|_U \equiv \{ F_U : U \rightarrow \mathbb R : f \in F \}F∣U​≡{FU​:U→R:f∈F}. Given two open sets U⊆WU \subseteq WU⊆W, we can restrict functions on WWW (a larger set) to functions on UUU (a smaller set). So we get maps F∣W→F∣UF|_W \rightarrow F|_UF∣W​→F∣U​. So given a function on a larger set WWW, we can inject into a smaller set UUU. But given a function on a smaller set, it's impossible to uniquely extend the function back into a larger set. These maps really are "one way". The reason it's a union of all functions is because we want to "identify" equivalent functions. We don't want to "take the product" of all germs of functions; We want to "take the union under equivalence".

§ Finite strings / A*

Given an alphabet set AAA, we can construct a finite limit of strings of length 000, strings of length 111, and so on for strings of any given length n∈Nn \in \mathbb Nn∈N. Here, the "problem" is that we can also find projection maps that allow us to "chop off" a given string, which makes this example not-so-great. However, this example is useful as it lets us contrast the finite and infinite string case. Here, we see that in the final limit A∗A*A∗, we will have all strings of finite length. (In the infinite strings case, which is an inverse limit, we will have all strings of infinite length)

§ Inverse limit: definition

An inverse limit consists of projections A1←A2←…A_1 \leftarrow A_2 \leftarrow \dotsA1​←A2​←…. It leads to a limit object LLL, which as a set is equal to a subset of the product of all the AiA_iAi​, where we only allow elements that "agree downwards" .Formally, we write this as:
L≡{a[:]∈∏iAi:proj(α←ω)(a[ω])=a[α] ∀α≤ω} L \equiv \{ a[:] \in \prod_i A_i : \texttt{proj}(\alpha \leftarrow \omega)(a[\omega]) = a[\alpha] ~ \forall \alpha \leq \omega \} L≡{a[:]∈i∏​Ai​:proj(α←ω)(a[ω])=a[α] ∀α≤ω}
So from each element in LLL, we get the projection maps that give us the component a[α]a[\alpha]a[α].
These 'feel like' cauchy sequences, where we are refining information at each step to get to the final object.

§ Inverse limit: prototypical example

§ infinite strings

We can consider the set of infinite strings. Given an infinite string, we can always find a finite prefix as a projection. However, it is impossible to canonically inject a finite prefix of a string into an infinite string! Given the finite string xxx, how do we make it into an infinite string? do we choose xxxa*, xxxb*, xxxc*, and so on? There's no canonical choice! Hence, we only have projections , but no injections .

§ P-adics

Consider the 7-adics written as infinite strings of digits in {0,1,…,6}\{0, 1, \dots, 6\}{0,1,…,6}. Formally, we start by:
  1. Having solutions to some equation in Z/7Z\mathbb{Z}/7\mathbb{Z}Z/7Z
  2. Finding a solution in Z/49Z\mathbb{Z}/49\mathbb{Z}Z/49Z that restricts to the same solution in Z/7Z\mathbb{Z}/7\mathbb{Z}Z/7Z
  3. Keep going.
The point is that we define the 777-adics by projecting back solutions from Z/49Z\mathbb{Z}/49\mathbb{Z}Z/49Z. It's impossible to correctly embed Z/7Z\mathbb{Z}/7\mathbb{Z}Z/7Z into Z/49Z\mathbb{Z}/49\mathbb{Z}Z/49Z: The naive map that sends the "digit i" to the "digit i" fails, because:
  • in Z/7Z\mathbb{Z}/7\mathbb{Z}Z/7Z we have that 2×4≡12 \times 4 \equiv 12×4≡1.
  • in Z/49Z\mathbb{Z}/49\mathbb{Z}Z/49Z 2×4≡82 \times 4 \equiv 82×4≡8.
So ϕ(2)×ϕ(7)≠ϕ(2×7)=ϕ(4)\phi(2) \times \phi(7) \neq \phi(2 \times 7) = \phi(4)ϕ(2)×ϕ(7)=ϕ(2×7)=ϕ(4). Hece, we don't have injections , we only have projections .

§ Partitions

Let SSS be some infinite set. Let {Πn}\{ \Pi_n \}{Πn​} be a sequence of partitions such that Πn+1\Pi_{n+1}Πn+1​ is finer than Πn\Pi_nΠn​. That is, every element of Πn\Pi_nΠn​ is the union of some elements of Πn+1\Pi_{n+1}Πn+1​. Now, given a finer partition, we can clearly "coarsen" it as desired, by mapping a cell in the "finer space" to the cell containing it in the "coarser space". The reverse has no canonical way of being performed; Once again, we only have projections , we have no injections . The inverse limit is:
{(P0,P1,P2,… )∈∏i=0nΠn:Pa=proja←z(Pz)∀a≤z}. \{ (P_0, P_1, P_2, \dots) \in \prod_{i=0}^n \Pi_n : P_a = \texttt{proj}_{a \leftarrow z}(P_z) \forall a \leq z \}. {(P0​,P1​,P2​,…)∈i=0∏n​Πn​:Pa​=proja←z​(Pz​)∀a≤z}.
But we only care about "adjacent consistency", since that generates the other consistency conditions; So we are left with:
{(P0,P1,P2,… )∈∏i=0nΠn:Pa=proja←b(Pb)∀a+1=b}. \{ (P_0, P_1, P_2, \dots) \in \prod_{i=0}^n \Pi_n : P_a = \texttt{proj}_{a \leftarrow b}(P_b) \forall a +1 = b \}. {(P0​,P1​,P2​,…)∈i=0∏n​Πn​:Pa​=proja←b​(Pb​)∀a+1=b}.
But unravelling the definition of proj\texttt{proj}proj, we get:
{(P0,P1,P2,… )∈∏i=0nΠn:Pa⊇Pb)∀a+1=b}. \{ (P_0, P_1, P_2, \dots) \in \prod_{i=0}^n \Pi_n : P_a \supseteq P_b) \forall a +1 = b \}. {(P0​,P1​,P2​,…)∈i=0∏n​Πn​:Pa​⊇Pb​)∀a+1=b}.
So the inverse limit is the "path" in the "tree of partitions".

§ Categorically

Categorically speaking, this is like some sort of product along with equating elements. This, cateogrically speaking, a inverse limit is a limit (recall that categorical limits exist iff products and equalizers exist).

§ Poetically, in terms of book-writing.

  • The direct limit is like writing a book one chapter after another. Once we finish a chapter, we can't go back, the full book will contain the chapter, and what we write next must jive with the first chapter. But we only control the first chapter (existential).
  • The inverse limit is like writing a book from a very rough outline to a more detailed outline. The first outline will be very vague, but it controls the entire narrative (universal). But this can be refined by the later drafts we perform, and can thus be "refined" / "cauchy sequence'd" into something finer.
