§ Krylov subspace method

This is a class of methods used to solve Ax=bAx = bAx=b, where AAA is sparse. Krylov subspace methods are a class of methods which use the idea of a Krylov subspace. There is conjugate gradient (CG), GMRES (Generalized minimal residual method).
