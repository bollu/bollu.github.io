§ Nakayama's lemma

I read the statement as IM=M  ⟹  M=0IM = M \implies M = 0IM=M⟹M=0, when III is in the jacobson radical.
  1. Essentially, it tells us that if a module MMM "lives by the III", then it also "dies by the III".
  2. Alternatively, we factor the equation as M(I−1)=0M(I - 1) = 0M(I−1)=0. Since our ideal IIIis a member of the jacobson radical, (1−I)(1 - I)(1−I) is "morally" a unit and thus M=0M = 0M=0. This is of course completely bogus, but cute nontheless.
  3. We can think of a graded ring, say R[x]R[x]R[x] acting on some graded module MMM (say, a subideal, M=(x2)M = (x^2)M=(x2)). When we compute IMIMIM, this will bump up the grading of MMM. If IM=MIM = MIM=M, then MMM could not have had non-trivial elements in the first place, since the vector of, say, "non-zero elements in each grade" which used to look like (v0,v1,v2,… )(v_0, v_1, v_2, \dots)(v0​,v1​,v2​,…) will now look like (0,v0,v1,… )(0, v_0, v_1, \dots)(0,v0​,v1​,…).
