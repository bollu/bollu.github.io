§ Weighted Burnside Lemma

  • I'm learning the weighted burnside lemma as a preamble to polya enumeration.
  • Define for a set XXX with a group action GGG, a weight function on the orbits OOO. Said differently, we have a weight function w:X→Ww: X \to Ww:X→W such that w(x)=w(g(x))w(x) = w(g(x))w(x)=w(g(x)) for all x∈Xx \in Xx∈X and g∈Gg \in Gg∈G.
