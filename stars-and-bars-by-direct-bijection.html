§ Stars and bars by direct bijection

We know that the number of kkk element multisets using letters from {1,…,n}\{1, \dots, n\}{1,…,n} is (k+n−1k)\binom{k+n-1}{k}(kk+n−1​). That is, we are allowed to pick elements from {1,…,n}\{1, \dots, n\}{1,…,n} repeatedly, and we want kkk such elements.

§ The usual proof: stars and bars

§ Direct bijection.

To build a direct bijection, map a kkk multiset of nnn into a kkk subset of n+k−1n+k-1n+k−1, which is counted by (n+k−1k)\binom{n+k-1}{k}(kn+k−1​).
  • We are first given a k=6k=6k=6 multiset of n=3n=3n=3, say m={3,1,2,1,3,3}m = \{3, 1, 2, 1, 3, 3\}m={3,1,2,1,3,3} ( mmm for multiset).
  • We make the representation unique by imposing an ascending order, so we write M=[1,1,2,3,3]M = [1, 1, 2, 3, 3]M=[1,1,2,3,3], where each M[i]≤M[i+1]M[i] \leq M[i+1]M[i]≤M[i+1].
  • Now, we map the above sequence to a set of unique values, by mapping N[i]=M[i]+iN[i] = M[i] + iN[i]=M[i]+i. Since M[i]≤M[i+1]M[i] \leq M[i+1]M[i]≤M[i+1] we have that M[i]+i<M[i+1]+(i+1)M[i] + i < M[i+1] + (i+1)M[i]+i<M[i+1]+(i+1).
  • This gives us the set M′={1+0,1+1,2+2,3+3,3+4}={1,2,3,6,7}M' = \{ 1+0, 1+1, 2+2, 3+3, 3+4 \} = \{ 1, 2, 3, 6, 7 \}M′={1+0,1+1,2+2,3+3,3+4}={1,2,3,6,7}.
  • See that this process is reversible.
