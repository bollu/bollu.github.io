§ Intuitionstic logic as a Heytig algebra

open sets form a Heytig Algebra, which is a lattice plus an implication operator. So it's stronger than a lattice, but weaker than a boolean algebra. Formally, a Heytig algebra over a set XXX is a collection (L,∨,∧,⇒)(L, \lor, \land, \Rightarrow)(L,∨,∧,⇒) where (X,∨,∧)(X, \lor, \land)(X,∨,∧) form a lattice, and ⇒\Rightarrow⇒ obeys the axiom
⇒:L→L;(c∧a)≤b  ⟺  c≤(a⇒b) \Rightarrow: L \rightarrow L; (c \land a) \leq b \iff c \leq (a \Rightarrow b) ⇒:L→L;(c∧a)≤b⟺c≤(a⇒b)
In any topological space (U,τ)(U, \tau)(U,τ) ( UUU for universe set) the open sets of that space form a Heytig algebra. Here, set-union and set-intersection become the lattice join and lattice meet. We define a "weak complement" denoted by ¬\lnot¬ which is defined as:

§ a -> b as a contained in b:

We'll show that a -> b is true/top/the full real line if and only if a is contained in b. We can show this using ¬a∨b\lnot a \lor b¬a∨b definition:
  1. a≤ba \leq ba≤b (given)
  2. Since a≤ba \leq ba≤b, ¬a≥¬b\lnot a \geq \lnot b¬a≥¬b, since ¬\lnot¬ reverses inclusion order.
  3. x≥yx \geq yx≥y implies that x∨p≥y∨px \lor p \geq y \lor px∨p≥y∨p for all p, since ppp is on both sides.
  4. Specializing to x=¬ax = \lnot ax=¬a, y=¬by = \lnot by=¬b, p=bp = bp=b gives us ¬a∨b≥¬b∨b\lnot a \lor b \geq \lnot b \lor b¬a∨b≥¬b∨b
  5. ¬b∨b\lnot b \lor b¬b∨b is universe UUU
  6. ¬a∨b≥U\lnot a \lor b \geq U¬a∨b≥U, which means it's equal to UUU (nothing can be greater than UUU).
We can also show this geometrically:

§ reading curry, uncurry using a -> b as containment:

curry, uncurry are the adjunction/isomorphism: 
((c,a) -> b) ~ (c -> (a -> b))
Read this as: 
( c     ,         a)     ->         b
(c `intersection` a) `contained in` b
if and only if 
c      ->        (a       ->      b)
c `contained in` (a `implication` b)
That is, we have "read" curry/uncurry as:
c∧a≤b  ⟺  c≤a⇒b c \land a \leq b \iff c \leq a \Rightarrow b c∧a≤b⟺c≤a⇒b
which was the definition of ⇒\Rightarrow⇒ we wanted!
