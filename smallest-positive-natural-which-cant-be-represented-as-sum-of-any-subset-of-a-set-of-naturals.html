§ Smallest positive natural which can't be represented as sum of any subset of a set of naturals

we're given a set of naturals S≡{xi}S \equiv \{ x_i \}S≡{xi​} and we want to find n≡min⁡{sum(T):T⊆S}n \equiv \min \{ sum(T): T \subseteq S \}n≡min{sum(T):T⊆S}, the smallest number that can't be written as a sum of elements from SSS.

§ By observation

  • Key observation: If we sort the set SSS as s[1]≤s[2]≤⋯≤s[n]s[1] \leq s[2] \leq \dots \leq s[n]s[1]≤s[2]≤⋯≤s[n], then we must have s[1]=1s[1] = 1s[1]=1. For if not, then 111 is the smallest nmber which cannot be written as a sum of elements.
  • Next, if we think about the second number, it must be s[2]=2s[2] = 2s[2]=2. If not, we return 222 as the answer.
  • The third number s[3]s[3]s[3] can be 333. Interestingly, it can also be 444, since we can write 3=1+23 = 1 + 23=1+2, so we can skip 333as an input.
  • What about s[4]s[4]s[4]? If we had [1≤2≤3][1 \leq 2 \leq 3][1≤2≤3] so far, then see that we can represent all numbers upto 666. If we have [1≤2≤4][1 \leq 2 \leq 4][1≤2≤4] so far, then we can represent all numbers upto 777. Is it always true that given a "satisfactory" sorted array AAA (to be defined recursively), we can always build numbers upto ∑A\sum A∑A?
  • The answer is yes. Suppose the array AAA can represent numbers upto ∑A\sum A∑A. Let's now append r≡sum(A)+1r \equiv sum(A)+1r≡sum(A)+1 into AAA. ( rrr for result). Define B := append(A, r). We claim we can represent numbers [1…(r+∑A)][1 \dots (r+\sum A)][1…(r+∑A)] using numbers from BBB. By induction hypothesis on A. We can represent [1⋯∑A][1 \dots \sum A ][1⋯∑A] from AAA. We've added r=∑A+1r = \sum A + 1r=∑A+1 to this array. Since we can build numbers [1⋯∑A][1\dots \sum A][1⋯∑A] from AAA, we can add rrr to this to build the range [r+1…r+∑A][r+1 \dots r + \sum A][r+1…r+∑A]. In total, by not choosing rrr, we build the segment [1⋯∑A][1 \dots \sum A ][1⋯∑A] and by choosing rrr we build the segment [∑A+1⋯∑A+r][\sum A + 1 \dots \sum A + r][∑A+1⋯∑A+r]giving us the full segment [1⋯∑A+r][1 \dots \sum A + r][1⋯∑A+r].
