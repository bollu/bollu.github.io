§ Modular inverse calculation

  • Easy way to calculate a−1a^{-1}a−1 mod ppp is to use ap−2a^{p-2}ap−2. We know that ap−1≡1a^{p - 1} \equiv 1ap−1≡1 from Lagrane's theorem, so ap−2⋅a≡1a^{p-2} \cdot a \equiv 1ap−2⋅a≡1, or a−1≡ap−2a^{-1} \equiv a^{p-2}a−1≡ap−2. This can be done fairly quickly with repeated exponentiation.
  • Another way to do this is to use extended eucliean division. Suppose a≢0 a \not \equiv 0a≡0 (mod p). Then we can find numbers α,β\alpha, \betaα,β such that aα+pβ=gcd(a,p)=1a \alpha + p \beta = gcd(a, p) = 1aα+pβ=gcd(a,p)=1. If we look at the whole equation (mod ppp), we find that aα≡1a \alpha \equiv 1aα≡1 (mod ppp), or α\alphaα is the modular inverse of aaa. 
pair<int, int> euc(int x, int y) {
  if (x < y) { return euc(y, x); }
  // x > y
  if (x % y == 0) { return {0, 1}; }
  int a, b; std::tie(a, b) = euc(y, x%y);
  // ay + b(x%y) = gcd(y, x%y) = gcd(x, y)
  // ay + b(x - y(x//y)) = gcd(y, x%y) = gcd(x, y)
  // ay + bx - by(x//y)  = gcd(y, x%y) = gcd(x, y)
  // (a - b(x//y))y + bx = gcd(y, x%y) = gcd(x, y)
  // bx + (a - b(x//y))y = gcd(y, x%y) = gcd(x, y)
  // intuition?
  return {b, a - b*(x/y)};
}
