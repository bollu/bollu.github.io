§ Stirling numbers and surjections

Interesting interpretation: The number of ways to surject an nnn element set into a kkk element set, since a surjection breaks a set up into a known number of fibers (in this case, kkk fibers).
This is not entirely true, because we only get kkk equivalence classes of the set {1,…,n}\{1, \dots, n\}{1,…,n}. We need to decide where to map each equivalence class. So the correct count of {[n]→onto[k]}\{ [n] \xrightarrow{onto} [k] \}{[n]onto​[k]} is k!S(n,k)k!S(n, k)k!S(n,k): there are k!k!k! ways to map equivalence classes of nnn to elements of kkk

§ Directly reading off the bijection between set partitions and rook placements

I found this very cool. The idea is to treat each rook as a "bouncer" that bounces light rays. All elements hit by a light ray belong to an equivalence class.
