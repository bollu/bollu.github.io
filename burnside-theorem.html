§ As local/global principle

See that the right-hand-side measures "local fixed points" in terms of ∣Stab(g)∣|Stab(g)|∣Stab(g)∣. The left hand side measures "global fixed points": an orbit O⊆XO \subseteq XO⊆X is a set such that G⋅O=OG \cdot O = OG⋅O=O. So it's a sort of "global fixed point of GGG". The burnside lemms tells us that we can recover the size of the set of global fixed points (the number of orbits) by averaging the size of the set of local fixed points (the average of the sizes of the stabilizers).
