§ Why commutator is important for QM

  • Suppose we have an operator LLL with eigenvector xxx, eigenvalue λ\lambdaλ. So Lx=λxLx = \lambda xLx=λx.
  • Now suppose we have another operator NNN such that [L,N]=κN[L, N] = \kappa N[L,N]=κN for some constant κ\kappaκ.
  • Compute [L,N]x=κNx[L, N]x = \kappa Nx[L,N]x=κNx, which implies:
[L,N]x=κNx(LN−NL)x=κNxL(Nx)−N(Lx)=κNxL(Nx)−N(λx)=κNxL(Nx)−λN(x)=κNxL(Nx)=κNx+λNxL(Nx)=(κ+λ)Nx \begin{aligned} &[L, N]x = \kappa Nx \\ &(LN - NL)x = \kappa Nx \\ &L(Nx) - N(Lx) = \kappa Nx \\ &L(Nx) - N(\lambda x) = \kappa Nx \\ &L(Nx) - \lambda N(x) = \kappa Nx \\ &L(Nx) = \kappa Nx + \lambda Nx \\ &L(Nx) = (\kappa + \lambda)Nx \\ \end{aligned} ​[L,N]x=κNx(LN−NL)x=κNxL(Nx)−N(Lx)=κNxL(Nx)−N(λx)=κNxL(Nx)−λN(x)=κNxL(Nx)=κNx+λNxL(Nx)=(κ+λ)Nx​
  • So NxNxNx is an eigenvector of LLL with eigenvalue κ+λ\kappa + \lambdaκ+λ.
  • This is how we get "ladder operators" which raise and lower the state. If we have a state xxx with some eigenvalue λ\lambdaλ, the operator like NNNgives us an "excited state" from xxx which eigenvalue κ+λ\kappa + \lambdaκ+λ.
