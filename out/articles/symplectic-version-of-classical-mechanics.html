§ Moment Map

Now that we have a method of going from a vector field XHX_HXH​ to a Hamiltonian HHH, we can go crazier with this. We can generate vector fields using Lie group actions on the manifold, and then look for hamiltonians corresponding to this lie group. This lets us perform "inverse Noether", where for a given choice of symmetry, we can find the Hamiltonian that possesses this symmetry.
