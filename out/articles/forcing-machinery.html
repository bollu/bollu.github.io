§ Ideal of a post

  • We define an ideal I⊆PI \subseteq PI⊆P to be a set of elements which are pairwise compatible (all pairs of elements have a union), and is downward closed (all elements with less information is present in the ideal).
  • More formally, for any i∈Ii \in Ii∈I and p∈Pp \in Pp∈P, if p≤ip \leq ip≤i, then p∈Ip \in Ip∈I. So P≤I  ⟹  P∈IP \leq I \implies P \in IP≤I⟹P∈I.

§ Density in a poset

  • A subset DDD of a poset PPP is dense iff for any p∈Pp \in Pp∈P, there is some d∈Dd \in Dd∈D such that d≥pd \geq pd≥p. Intuitively, at any point in the poset, it is possible to "add more information" to reach DDD.

§ Fundamental theorem of forcing

  • For every formula ϕ\phiϕ, for every generic ideal GGG over PPP:
  • 1. Definability: there is a set F(α,ϕ)∈MF(\alpha, \phi) \in MF(α,ϕ)∈M such that p⊩ϕ(τ⃗)p \Vdash \phi(\vec \tau)p⊩ϕ(τ) ( ppp forces τ\tauτ) if and only if (p,τ⃗)∈F(α,ϕ)(p, \vec \tau) \in F(\alpha, \phi)(p,τ)∈F(α,ϕ). That is, the forcing relation is definable in MMM
  • 2. Completeness: ϕM[G](τ⃗G)\phi^{M[G]}(\vec \tau^G)ϕM[G](τG) is true iff there is a g∈Gg \in Gg∈G such that g⊩ϕ(τ⃗)g \Vdash \phi(\vec \tau)g⊩ϕ(τ). That is, any true sentence in M[G]M[G]M[G] must have a witnessing p∈Gp \in Gp∈G which forces it, for any generic ideal GGG.
  • 3. Coherence/Stability: If p⊩ϕp \Vdash \phip⊩ϕ, for all q≥pq \geq pq≥p, we have q⊩ϕq \Vdash \phiq⊩ϕ. Truth once forced cannot be unforced, truth is inflationary, truth is stable, etc.
  • The FTF (fundamental theorem of forcing) is an algorithm on the ZFC syntax. It takes a formula ϕ\phiϕ, and produces a ZFC proof of (1), (2), (3).
